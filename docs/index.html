How it works

Only talk after you add them

Search by username, add the person, then start a private thread. The app does not expose a public follower graph or broad contact feed.

Calling

Voice and video built in

One-to-one voice and video calling run inside the same conversation flow, with notification actions for accepting or declining incoming calls.

Notifications

Generic by default

Message and call notifications use system channels and keep their text generic, so the lock screen reveals less about who is contacting you.
